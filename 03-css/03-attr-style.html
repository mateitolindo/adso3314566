03 - Attr Style

Inline Styles Method

En este ejemplo, estamos aplicando el diseño mediante el atributo style dentro de cada etiqueta. Aunque funciona, hace que el HTML sea difícil de leer y mantener.

<h1 style="color: blue;">Hola Mundo</h1>

*Nota: Los estilos inline tienen la mayor especificidad, ganándole a los estilos de archivos externos.
